Projet CRUD PHP

Bienvenue dans votre projet. Ce projet utilise PHP et MySQL.

Note importante pour VS Code :
Vous ne pouvez pas exécuter les fichiers PHP directement en double-cliquant dessus ou en utilisant l'extension "Live Server".

Vous devez installer un serveur local (comme XAMPP, WAMP ou MAMP), placer ce dossier dans le dossier htdocs (ou www), et y accéder via votre navigateur à l'adresse http://localhost/projet_crud_php.
